# Thapelo Moloto Portfolio

Single-page portfolio website for Thapelo Moloto, focused on senior data engineering, SQL architecture, and legacy modernisation.

## Live Site

After GitHub Pages is enabled, the site URL will be:

https://tshiamotlm.github.io/thapelo_tshiamo-portfolio/

## Tech Stack

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript

## Local Preview

Open `index.html` directly in your browser, or use any static server.

## Deployment (GitHub Pages)

1. Push changes to the `main` branch.
2. In GitHub repository settings, open Pages.
3. Set Source to "Deploy from a branch".
4. Select branch `main` and folder `/ (root)`.
5. Save.

## Update Workflow

1. Edit files locally.
2. Commit and push:

   git add .
   git commit -m "Update portfolio"
   git push

GitHub Pages will redeploy automatically on each push.
